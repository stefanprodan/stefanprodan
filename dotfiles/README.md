# Dev machine macOS setup

## Homebrew

Install Homebrew:

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

Install tools

```bash
brew bundle --file=dotfiles/Brewfile
```
